
# 🚀 IntelliJ IDEA 常用快捷键大全（Windows/Linux & macOS 对照）

在日常开发中，熟练掌握快捷键能显著提升效率，减少鼠标操作。本文整理了 IntelliJ IDEA 常用快捷键，以表格形式呈现，涵盖 **搜索、编辑、导航、重构、调试、项目窗口**等常见场景，方便速查与学习。

---

## 🔍 全局搜索与查找

| 功能描述       | Windows/Linux | macOS |
|----------------|---------------|-------|
| 搜索一切       | 双击 Shift    | 双击 Shift |
| 查找类         | Ctrl + N      | Cmd + O |
| 查找文件       | Ctrl + Shift + N | Cmd + Shift + O |
| 查找符号       | Ctrl + Alt + Shift + N | Cmd + Option + O |
| 查找操作       | Ctrl + Shift + A | Cmd + Shift + A |

---

## 🚀 代码编辑

| 功能描述       | Windows/Linux | macOS |
|----------------|---------------|-------|
| 完成当前语句   | Ctrl + Shift + Enter | Cmd + Shift + Enter |
| 格式化代码     | Ctrl + Alt + L | Cmd + Alt + L |
| 复制当前行     | Ctrl + D      | Cmd + D |
| 删除当前行     | Ctrl + Y      | Cmd + Backspace |
| 上下移动行     | Alt + Shift + ↑/↓ | Alt + Shift + ↑/↓ |
| 扩展/缩小选择范围 | Ctrl + W / Ctrl + Shift + W | Alt + ↑ / Alt + ↓ |
| 注释/取消注释   | Ctrl + / / Ctrl + Shift + / | Cmd + / / Cmd + Shift + / |

---

## 🧭 导航跳转

| 功能描述       | Windows/Linux | macOS |
|----------------|---------------|-------|
| 转到声明处     | Ctrl + B      | Cmd + B |
| 转到实现处     | Ctrl + Alt + B | Cmd + Alt + B |
| 最近打开文件   | Ctrl + E      | Cmd + E |
| 最近编辑位置   | Ctrl + Shift + E | Cmd + Shift + E |
| 错误间跳转     | F2 / Shift + F2 | F2 / Shift + F2 |
| 后退/前进导航  | Ctrl + Alt + ←/→ | Cmd + Alt + ←/→ |

---

## 🛠️ 重构代码

| 功能描述       | Windows/Linux | macOS |
|----------------|---------------|-------|
| 显示重构菜单   | Ctrl + Alt + Shift + T | Ctrl + T |
| 重命名         | Shift + F6    | Shift + F6 |
| 提取方法       | Ctrl + Alt + M | Cmd + Alt + M |
| 提取变量       | Ctrl + Alt + V | Cmd + Alt + V |
| 内联           | Ctrl + Alt + N | Cmd + Alt + N |
| 安全删除       | Alt + Delete  | Cmd + Delete |

---

## 🐞 调试

| 功能描述       | Windows/Linux | macOS |
|----------------|---------------|-------|
| 单步进入       | F7            | F7 |
| 单步跳过       | F8            | F8 |
| 恢复运行       | F9            | Cmd + Option + R |

---

## 📁 项目窗口

| 功能描述       | Windows/Linux | macOS |
|----------------|---------------|-------|
| 打开/隐藏 Project 面板 | Alt + 1 | Cmd + 1 |
| 打开/隐藏 Console 控制台 | Alt + 4 | Cmd + 4 |
| 打开类结构     | Alt + 7       | Cmd + 7 |

---

## 📌 总结

- IDEA 的快捷键可以在 **File → Settings → Keymap** 中自定义。  
- 建议开发者逐步养成使用快捷键的习惯，尤其是 **搜索、编辑、导航** 三大类，能显著提升效率。  
- 如果你在写博客或做笔记，可以将这些表格直接嵌入，作为速查手册。
