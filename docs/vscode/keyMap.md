

# VS Code 常用快捷键总表

| 功能                     | Windows / Linux        | macOS                  |
|--------------------------|------------------------|------------------------|
| 打开/关闭终端            | `Ctrl + \``            | `Cmd + \``             |
| 切换底部面板（终端/调试）| `Ctrl + J`             | `Cmd + J`              |
| 删除当前行               | `Ctrl + Shift + K`     | `Cmd + Shift + K`      |
| 向下复制当前行           | `Shift + Alt + ↓`      | `Option + Shift + ↓`   |
| 向上复制当前行           | `Shift + Alt + ↑`      | `Option + Shift + ↑`   |
| 向下移动当前行           | `Alt + ↓`              | `Option + ↓`           |
| 向上移动当前行           | `Alt + ↑`              | `Option + ↑`           |
| 复制整行并粘贴到其他位置 | `Ctrl + C / Ctrl + V`  | `Cmd + C / Cmd + V`    |
| 跳转到 import 引用文件   | `Ctrl + 左键点击`      | `Cmd + 左键点击`       |
| 快速打开文件             | `Ctrl + P`             | `Cmd + P`              |
| 返回上一个位置           | `Alt + ←`              | `Ctrl + -`             |
| 前进到下一个位置         | `Alt + →`              | `Ctrl + Shift + -`     |
